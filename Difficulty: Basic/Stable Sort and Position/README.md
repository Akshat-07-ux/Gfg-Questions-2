<h2><a href="https://www.geeksforgeeks.org/problems/stable-sort-and-position1359/1?page=9&category=Sorting&sortBy=submissions">Stable Sort and Position</a></h2><h3>Difficulty Level : Difficulty: Basic</h3><hr><div class="problems_problem_content__Xm_eO" style="--text-color: var(--problem-text-color);"><p><span style="font-size: 18.6667px;">Given an&nbsp;</span><span style="font-size: 18.6667px;">integer</span><span style="font-size: 18.6667px;">&nbsp;array </span><strong style="font-size: 18.6667px;">arr[]</strong><span style="font-size: 18.6667px;">, that may contain duplicate elements and an index </span><strong style="font-size: 18.6667px;">k </strong><span style="font-size: 18.6667px;">(0-based), find the final position (0-based) of the element at index</span><strong style="font-size: 18.6667px;"> k</strong><span style="font-size: 18.6667px;"> after applying a stable sort on the array.</span></p>
<p><span style="font-size: 18.6667px;"><strong>Note</strong>: In a stable sort, elements with equal values retain their relative order from the original array.</span></p>
<p><span style="font-size: 14pt;"><strong>Examples:</strong></span></p>
<pre><span style="font-size: 14pt;"><strong>Input:</strong> arr[] = [3, 4, 3, 5, 2, 3, 4, 3, 1, 5], k = 5
<strong>Output:</strong> 4
<strong>Explanation</strong>: The element at index 5 is 3. There are 2 elements smaller than 3, so the group of 3s starts at position 2. <br>Among all occurrences of 3 (at indices 0, 2, 5, 7), index 5 is the 3rd one (0-based rank 2). <br>Final position = 2 + 2 = 4.</span></pre>
<pre><span style="font-size: 14pt;"><strong>Input:</strong> arr[]= [3, 4, 3, 5, 2, 3, 4, 3, 1, 5], k = 2
<strong>Output:</strong> 3
<strong>Explanation</strong>: The element at index 2 is 3. There are 2 elements smaller than 3, so the group of 3s starts at position 2. <br>Among all occurrences of 3 (at indices 0, 2, 5, 7), index 2 is the 2nd one (0-based rank 1). <br>Final position = 2 + 1 = 3.</span></pre>
