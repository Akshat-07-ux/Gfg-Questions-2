<h2><a href="https://www.geeksforgeeks.org/problems/overlapping-intervals--174556/1?page=9&category=Sorting&sortBy=submissions">Overlapping Intervals</a></h2><h3>Difficulty Level : Difficulty: Easy</h3><hr><div class="problems_problem_content__Xm_eO" style="--text-color: var(--problem-text-color);"><p><span style="font-size: 18.6667px;">You are given a 2D array <strong>arr[][] </strong>which represents a set of intervals, where each element <strong>arr[i] = [start, end]</strong> defines an interval</span><span style="font-size: 18.6667px;">. <br></span><span style="font-size: 18.6667px;">Your task is to determine if any two intervals in the given set <strong>overlap</strong>.</span></p>
<p><span style="font-size: 18.6667px;"><strong>Note:</strong> Two intervals [a, b] and [c, d] overlap if they have at least one common value, i.e., a ≤ d and c ≤ b.</span></p>
<p><span style="font-size: 14pt;"><strong>Examples:</strong></span></p>
<pre><span style="font-size: 14pt;"><strong>Input: </strong>n = 4, arr[][] = [[1, 3], [5, 7], [2, 4], [6, 8]]
<strong>Output: </strong>true<strong>
Explanation:</strong> The intervals [1, 3] and [2, 4] overlap.</span></pre>
<pre><span style="font-size: 14pt;"><strong>Input: </strong>n = 4, arr[][] = [[1, 3], [7, 9], [4, 6], [10, 13]]
<strong>Output: </strong>false<strong>
Explanation:</strong> No pair of intervals overlap.</span></pre>
