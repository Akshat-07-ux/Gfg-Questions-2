<h2><a href="https://www.geeksforgeeks.org/problems/minimize-cash-flow/1?page=6&category=Greedy&sortBy=submissions">Minimize Cash Flow</a></h2><h3>Difficulty Level : Difficulty: Medium</h3><hr><div class="problems_problem_content__Xm_eO" style="--text-color: var(--problem-text-color);"><p><span style="font-size: 18px;">Given <strong>n</strong> friends and a 2D array <strong>transaction[][]</strong>, where transaction[i][j] denotes the amount of money that friend i owes to friend j .</span><span style="font-size: 18px;"> Design an algorithm by which the total cash flow among all the friends is minimized.&nbsp;</span></p>
<p><strong><span style="font-size: 18px;">Note:&nbsp;</span></strong><span style="font-size: 18px;">Please note there can be multiple solutions possible, and the solution will be judged according to its net flow, and if it correctly follows the cash flow. If, you're solution returns -1, this means the cash flow is not following the actual flow of cash.</span></p>
<p><strong><span style="font-size: 18px;">Examples:</span></strong></p>
<pre><span style="font-size: 18px;"><strong>Input: </strong>transaction[][] = [[0, 100, 0], [0, 0, 100], [100, 0, 0]]<br><strong>Output:</strong> [[0, 0, 0], [0, 0, 0], [0, 0, 0]]<br><span style="font-size: 14pt;"><strong style="font-family: -apple-system, BlinkMacSystemFont, 'Segoe UI', Roboto, Oxygen, Ubuntu, Cantarell, 'Open Sans', 'Helvetica Neue', sans-serif;">Explanation: </strong></span>Since friend one has to give friend two which has to give friend three and which in turn has to give one. So it is better than no one will do anything to anyone.</span></pre>
<pre><span style="font-size: 18px;"><strong>Input: </strong>transaction[][] = [[0, 100, 0], [0, 0, 200}, [0, 0, 0]]
<strong>Output: </strong>[[0, 0, 100], [0, 0, 100], [0, 0, 0]]
<strong>Explanation: </strong>The net flow is minimized.</span></pre>
