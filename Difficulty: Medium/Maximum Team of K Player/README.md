<h2><a href="https://www.geeksforgeeks.org/problems/maximum-team-of-k-player/1?page=5&category=Searching&sortBy=submissions">Maximum Team of K Player</a></h2><h3>Difficulty Level : Difficulty: Medium</h3><hr><div class="problems_problem_content__Xm_eO"><p><span style="font-size: 18px;">Given an array&nbsp;<strong>arr[]</strong>&nbsp;consisting of&nbsp;<strong>n</strong>&nbsp;positive integers and a &nbsp;positive integer&nbsp;<strong>k</strong>&nbsp;such that there are&nbsp;<strong>n</strong>&nbsp;countries, each country has&nbsp;<strong>arr[i]</strong> players. Your task is to find the <strong>maximum</strong> number of teams that can be formed by forming teams of size <strong>k</strong>&nbsp;such that each player in the team is from a different country.</span></p>
<p><span style="font-size: 18px;"><strong>Examples:</strong></span></p>
<pre><span style="font-size: 18px;"><strong>Input:</strong> arr[] = [4, 3, 5, 3], k = 3
<strong>Output: </strong>5
<strong>Explaination: </strong>Consider the countries are named A, B, C and D. The possible ways of forming the teams are [A, B, C], [A, C, D], [A, B, C], [B, C, D], [A, C, D] such that in each set there are no more than 1 person from a country.Therefore, the total count teams formed is 5.</span></pre>
<pre><span style="font-size: 18px;"><strong>Input: </strong>arr[] = [2, 3, 4], k = 2
<strong>Output: </strong>4
<strong>Explaination: </strong>Consider the countries are named A, B, C and D. The possible ways of forming the teams are {B, C}, {B, C}, {A, C}, ({A, B} or {A, C} or {B, C}) such that in each set there are no more than 1 person from a country.Therefore, the total count teams formed is 4.</span></pre>
<p><span style="font-size: 18px;"><strong>Constraints:</strong></span><br><span style="font-size: 18px;">1 ≤ n ≤ 10<sup>5</sup></span><br><span style="font-size: 18px;">1 ≤ k ≤ n</span><br><span style="font-size: 18px;">1 ≤ arr[i] ≤ 10<sup>4</sup></span></p></div><p><span style=font-size:18px><strong>Company Tags : </strong><br><code>Google</code>&nbsp;<br><p><span style=font-size:18px><strong>Topic Tags : </strong><br><code>Arrays</code>&nbsp;<code>Searching</code>&nbsp;<code>Mathematical</code>&nbsp;<code>Binary Search</code>&nbsp;<code>Data Structures</code>&nbsp;<code>Algorithms</code>&nbsp;
